# Sesion guiada de usabilidad

Este documento esta pensado para usarlo tal cual con 3 usuarios de prueba. Solo tienes que abrir la app, darles una consigna y anotar lo que ocurra.

## Objetivo

Comprobar si la interfaz resulta clara para tres perfiles distintos sin necesidad de explicarles cada paso.

## Duracion recomendada

- 8 a 12 minutos por persona.

## Regla importante

No les guies en mitad de la tarea salvo que se bloqueen por completo. Lo que interesa es ver si la interfaz se explica sola.

## Preparacion previa

Antes de empezar:

1. Carga la app con datos demo limpios.
2. Ten listas credenciales de prueba.
3. Decide si haras la sesion en movil, tablet o web.
4. Anota dispositivo y tamano aproximado.

## Usuario 1: perfil oficina o administrador

### Contexto

“Imagina que acabas de entrar para revisar la operativa del dia y localizar el modulo de partes.”

### Tareas

1. Inicia sesion.
2. Identifica en que area de la app has entrado.
3. Abre el modulo `Partes`.
4. Localiza si hay un trabajo pendiente o una accion visible relacionada con partes.
5. Vuelve a la pantalla anterior o al panel principal.

### Que observar

- Si entiende rapido el menu principal.
- Si diferencia panel, partes, flota y equipo.
- Si encuentra la forma de volver sin perderse.

## Usuario 2: perfil operario o trabajador

### Contexto

“Imagina que vienes a fichar y revisar tu trabajo del dia.”

### Tareas

1. Inicia sesion.
2. Encuentra `Fichaje`.
3. Inicia una jornada.
4. Indica el tipo de jornada.
5. Identifica donde se cierra la jornada o donde se ve el historico.
6. Abre despues el modulo `Partes`.

### Que observar

- Si entiende la accion principal de fichar.
- Si la seleccion de jornada resulta clara.
- Si distingue entre fichaje, partes y ausencias.

## Usuario 3: perfil comercial o cliente

Elige una de estas dos variantes segun el perfil que tengas disponible.

### Variante comercial

Contexto:

“Imagina que necesitas revisar un presupuesto y comprobar que todo esta en orden.”

Tareas:

1. Inicia sesion.
2. Encuentra `Presupuestos`.
3. Abre uno.
4. Di en voz alta que accion harías a continuacion.
5. Encuentra como volver.

### Variante cliente

Contexto:

“Imagina que entras para revisar un presupuesto o consultar tus embarcaciones.”

Tareas:

1. Inicia sesion.
2. Encuentra `Presupuestos` o `Flota`.
3. Abre la seccion correcta.
4. Encuentra `Cuenta` o la opcion de cerrar sesion.

### Que observar

- Si entiende la navegacion lateral o el drawer.
- Si identifica rapidamente la seccion que necesita.
- Si ve claro que esta en un area distinta al admin o al operario.

## Preguntas finales para cada usuario

Haz estas tres preguntas al terminar:

1. ¿Que parte te ha parecido mas clara?
2. ¿En que momento has dudado mas?
3. ¿Que cambiarias para que fuese mas facil?

## Consejo practico

Si no te da tiempo a hacer 5 usuarios, haz 3 bien hechos. Para esta rubrica vale mas una prueba corta, real y documentada que una descripcion inventada o demasiado generica.
