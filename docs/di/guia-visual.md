# Guia visual y coherencia de interfaz

## 1. Direccion visual

NavalGO sigue una guia propia coherente inspirada en un contexto naval profesional:

- gama principal azul marino, turquesa, arena y coral;
- superficies claras con contraste alto;
- tipografia `Manrope` para tono tecnico y moderno;
- tarjetas, paneles y chips para lectura operativa rapida.

## 3. Principios de interfaz

- Prioridad a tareas frecuentes.
- Informacion agrupada por paneles.
- Jerarquia clara entre accion primaria y secundaria.
- Menus persistentes por rol.
- Estados visibles mediante chips, iconos y color.

## 4. Coherencia entre app y landing

- Misma marca `NavalGO`.
- Misma familia de color.
- Misma tipografia principal.
- Mensaje de producto alineado: control operativo, flota, partes y evidencias.

## 5. Justificacion para la rubrica DI

No se ha seguido una guia oficial cerrada de un sistema externo, sino una guia propia consistente aplicada a:

- app Flutter;
- componentes reutilizables;
- web comercial;
- documentacion de producto.

## 6. Checklist visual antes de capturar

- Sin textos con codificacion rota.
- Sin overflows ni recortes.
- Sin pantallas vacias o provisionales.
- Con datos demo limpios y creibles.
- Con contraste suficiente en textos secundarios.
