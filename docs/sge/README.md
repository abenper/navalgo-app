# SGE

Esta carpeta reune la documentacion tecnica y operativa del sistema NavalGO
para la rubrica de Sistemas de Gestion Empresarial. El objetivo es cubrir todos
los modulos, su despliegue, los accesos, la trazabilidad y la operacion real.

## Resumen rapido

- Cubre frontend Flutter, backend Spring Boot, landing comercial y despliegue.
- Incluye diagramas de modulos, clases y flujos principales.
- Documenta requisitos de sistema, variables, base de datos y despliegue.
- Explica consultas, acceso por roles, manipulacion y exportacion de datos.
- Reune mecanismos de autoria, trazabilidad e incidencias reales del proyecto.
