# Entrega Final

Este bloque reúne el material final para preparar la defensa y congelar una versión estable de NavalGO sin improvisar el último día.

## Contenido

- [capturas-entrega.md](./capturas-entrega.md): inventario de capturas, estado actual y tomas pendientes.
- [guion-demo-5-7-min.md](./guion-demo-5-7-min.md): demo corta con orden, tiempos y frases de apoyo.
- [guion-defensa.md](./guion-defensa.md): defensa oral centrada en problema, solución, arquitectura, pruebas y seguridad.
- [version-estable.md](./version-estable.md): checklist de congelación de entrega.
- `capturas/`: capturas ya copiadas al repo para no depender del escritorio.

## Objetivo

La idea es que puedas:

- preparar una memoria visual coherente;
- grabar una demo de 5 a 7 minutos sin saltos;
- defender el proyecto con un discurso claro;
- llegar a la entrega con una versión verificada y estable.
